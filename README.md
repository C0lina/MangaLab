# MangáLab - Um site de leitura livre de mangás

# MangáLab é um site de leitura livre de mangás, onde os usuários podem acessar uma vasta coleção de mangás de forma gratuita. O site oferece uma interface amigável e fácil de usar, permitindo que os leitores desfrutem de seus mangás favoritos sem a necessidade de pagar por eles.

## 🛠️ Construído com

Aqui as ferramentas e bibliotecas usadas para construir o projeto:

* [MongoDb](https://www.mongodb.com/) - Utilizado para o banco de dados
* [JavaScript](https://developer.mozilla.org/pt-BR/docs/Web/JavaScript) - Utilizado para o frontend e backend
